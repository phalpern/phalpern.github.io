If you have reached this page, then you are authorized to download a demo of CLI Power Tools. The available downloads are as follows:

[clidemo-Linux-x86.tar.gz] Linux for Intel x86 processors.
(OLD) [clidemo-w32.exe] Windows 32 (self-extracting archive)

Unpacking the above file will create the following directory structure:

clidemo/ Top-level directory
clidemo/bin/clidemo[.exe] CLI demonstration program executable file.
clidemo/cdl CDL files used to create the demo.
clidemo/include Misc. include files used to create the demo
clidemo/src Misc. source files used to create the demo
clidemo/cdlout Files generated by the CDL compiler (.hh and .cc).
